ivy:validateProject

Full name:

com.axonivy.ivy.ci:project-build-plugin:14.0.0-jakarta-SNAPSHOT:validateProject

Description:

Validates an ivy Project.

Attributes:

  • Requires a Maven project to be executed.
  • Requires dependency resolution of artifacts in scope: compile.
  • The goal is thread-safe and supports parallel builds.

Optional Parameters

Name Type Since Description
<excludeValidators> List<String> 14.0.0 List of validators to deliberately exclude from the validation run. The following are currently included:
  • role
  • database
  • form
  • dataclass
  • process
  • restclient
  • webserviceclient
  • variable
  • user
Configure a fixed set in the POM: 
<configuration>
<excludeValidators>
<validator>process</validator>
<validator>role</validator>
</excludeValidators>
</configuration>
Or set it directly from the command line as a comma-separated list, without touching the POM: 
mvn install -Divy.validation.excludeValidators=role,webservice

User Property: ivy.validation.excludeValidators
<skipProjectValidation> boolean 14.0.0 Set to true to completely skip the project validation.
Default: false
User Property: ivy.validation.skip
<skipScriptValidation> boolean 8.0.3
Deprecated.
use skipProjectValidation (property ivy.validation.skip) instead, which skips the validation entirely rather than just the script validation part. This parameter will be removed in a future version.

Set to false to perform the validation of ivyScript code within ivy processes.
Default: false
User Property: ivy.script.validation.skip
