ivy:ivy-add-resource

Full name:

com.axonivy.ivy.ci:project-build-plugin:14.0.0-SNAPSHOT:ivy-add-resource

Description:

Dynamically adds resource directories to the Maven build. All direct subdirectories of src_generated/rest/ and src_generated/ws/ are added as resource directories.

Attributes:

  • Requires a Maven project to be executed.
  • The goal is thread-safe and supports parallel builds.
  • Since version: 14.0.0.
  • Binds by default to the lifecycle phase: generate-resources.

Optional Parameters

Name Type Since Description
<skipIvyAddResource> boolean - Set to true to skip this mojo entirely.
Default: false
User Property: ivy.add.resource.skip
