ivy:generate-dialog-form-sources

Full name:

com.axonivy.ivy.ci:project-build-plugin:14.0.0-SNAPSHOT:generate-dialog-form-sources

Description:

Generates Dialog Form source files.

Command line invocation is supported.

mvn com.axonivy.ivy.ci:project-build-plugin:generate-dialog-form-sources

Source generation can be skipped using:

mvn com.axonivy.ivy.ci:project-build-plugin:generate-dialog-form-sources
-Divy.generate.dialog.form.sources.skip=true

Attributes:

  • Requires a Maven project to be executed.
  • The goal is not marked as thread-safe and thus does not support parallel builds.
  • Since version: 13.2.0.

Optional Parameters

Name Type Since Description
<skipGenerateSources> boolean 13.2.0 Set to true to bypass the generation of ivy dialog form sources.
Default: false
User Property: ivy.generate.dialog.form.sources.skip
