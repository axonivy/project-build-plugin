ivy:ivy-cleanup

Full name:

com.axonivy.ivy.ci:project-build-plugin:13.2.0-SNAPSHOT:ivy-cleanup

Description:

Deletes directories that contain generated sources. By default, the following directories are removed: lib/mvn-deps, src_dataClasses, src_wsproc, and src_generated.

Attributes:

  • Requires a Maven project to be executed.
  • The goal is not marked as thread-safe and thus does not support parallel builds.
  • Since version: 13.2.0.

Optional Parameters

Name Type Since Description
<cleanupExcludes> String[] - Define directories to be excluded from cleanup with ANT-style exclusions. Sample: 
<cleanupExcludes>
  <cleanupExclude>src_dataClasses</cleanupExclude>
  <cleanupExclude>src_wsproc</cleanupExclude>
</cleanupExcludes>
<cleanupIncludes> String[] - Define directories to be deleted during cleanup with ANT-style inclusions. Sample: 
<cleanupIncludes>
  <cleanupInclude>custom/path</cleanupInclude>
</cleanupIncludes>
<skipIvyDependencyCleanup> boolean - Set to true to bypass the deletion step.
Default: false
User Property: ivy.cleanup.skip
