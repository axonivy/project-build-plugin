project-build:ivy-test-properties

Full name:

com.axonivy.ivy.ci:project-build-plugin:7.4.0-SNAPSHOT:ivy-test-properties

Description:

Shares the classpath of the built ivy project and it's engine as public property and tries to auto-configure maven-surefire-plugin to use this classpath.

Attributes:

  • Requires a Maven project to be executed.
  • Since version: 6.0.2.

Optional Parameters

Name Type Since Description
skipTest boolean 6.1.0 Set to true to bypass property set-up.
Default value is: false.
User property is: maven.test.skip.

Parameter Details
